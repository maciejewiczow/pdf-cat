# pdf-cat
## Installation steps:
1. ```bash
   git cone https://github.com/maciejewiczow/pdf-cat.git
   ```
1. ```bash
   npm install
   ```
1. ```bash
   npm link
   ```

## Usage
```bash
pdf-cat [Input Files] -o [output_file]
```
or
```bash
pdf-cat [Input Files] > [output_file]
```
or
```bash
pdf-cat [Input Files] | [other program]
```
