# pdf-cat
Simple CLI utility, used to concatenate pdf files

## Installation steps:
1. `git cone https://github.com/maciejewiczow/pdf-cat.git`
1. `npm install`
1. `npm link`

## Usage
```bash
pdf-cat [Input Files] -o [output_file]
```
or
```bash
pdf-cat [Input Files] > [output_file]
```
or
```bash
pdf-cat [Input Files] | [other program]
```
